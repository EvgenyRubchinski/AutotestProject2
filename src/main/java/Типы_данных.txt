boolean- хранит значение true\false
byte - число от -128 до 127 и занимает 1 байт
short - целое число от -32768 до 32767 и занимает 2 байта
int - целое число от -2147483648 до 2147483647 и занимает 4 байта
long - целое число от -9.......08 до 9.....807
double - число с плавающей точкой(дробные числа)
float - число с плавающей точкой
char- одиночный символ в кодировке, числовое значение от 0 до 65535


строковые переменные представляют собой объект String
с помощью операций + можно присоеденить к одной строке другую.
для переноса на след строку, применяется последовательность \n

String text = "Для хнанения данных в программе преднозначены переменные. \n"+
     "каждая переменная имеет тип, имя и значение";
System.out.println(text);
